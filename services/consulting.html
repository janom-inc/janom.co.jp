暗号通貨・仮想通貨関連

国内でも最高峰の暗号通貨の知識や実務経験をもとに、ブロックチェイン等を用いた新規事業の創出・既存プロダクトのセキュリティ強化・既存業務の効率改善等を行います。

暗号通貨・仮想通貨プロダクト開発支援

Bitcoin や Ripple、ブロックチェインを用いた新規事業創出・業務改善等の支援を行います。

  • 暗号通貨・仮想通貨・ブロックチェイン技術を用いた実証実験の提案および実施
  • 暗号通貨関連事業の立ち上げに際し、最適な設計（スケーラブル、軽量、高い拡張性等）の提案および制作
  • コイン盗難防止等セキュリティ対策の提案および実施
暗号通貨・仮想通貨レクチャー

暗号通貨やブロックチェインの基本的な仕組みについてのレクチャーや最新技術解説等を行い、御社技術者の知識向上に対する支援を行います。

  • Bitcoin 原論文の解説（参考スライド資料（古））
  • Bitcoin のウォレット機能（入金・出金）の実装方法についてのレクチャー
  • Lightning Network 等の Bitcoin スケーラビリティ改善案の技術解説
  • セキュリティ（ECDSAやハッシュ関数の安全性等）、プライバシー（送金者や送金額の秘匿化技術等）に関する最新動向解説（参考スライド資料）

コンピューティング最適化・高性能計算

御社ソフトウェアで利用されている既存処理コードの最適化・アルゴリズム改善・スーパーコンピュータ（スパコン）や GPU クラスタを用いた計算処理最適化などを行います。

コード・アルゴリズム最適化

御社ソフトウェアで利用されている、もしくは利用を検討中のコードやアルゴリズムに対し、計算処理の実行を行うハードウェアアーキテクチャにあわせた最適化を行い、計算能力を最大限まで引き出します。これにより、計算時間の短縮によるユーザビリティ向上・サーバ負荷軽減によるコスト削減効果等が追加のハードウェア投資なしで実現できます。

  • 利用アルゴリズムの再検討。効率的な代替アルゴリズムの提案および実装を行います
  • コードのプロファイリングと最適化。マルチコア対応、SIMD化、キャッシュ利用効率向上等を実施し、追加のハードウェア投資なしでスピードアップを図ります
スパコンやGPUへの対応

計算処理の、スパコンやクラスタ・GPUへの対応を行います。これにより従来の計算時間と比べて最大で数十倍ものスピードアップが可能となり、研究開発サイクルの短縮・ユーザビリティ向上等が実現できます。Amazon Web Service (AWS) などを用いることで、自社でハードウェアを持たないことによるコスト削減のご提案も可能です。

  • 既存コードのスパコン・クラスタ・GPU対応。メニーコアプログラミングの領域では、長年の知識や経験なしでは効率の良いコードは書けません。熱統計物理学の数値計算研究を行ってきた経験をもとに、計算を実行するハードウェアにあわせた最適化を行い、その性能を最大限まで引き出します
  • GPGPU 処理コードの最適化。単純に GPU で計算を行うようにしただけでは、多くの場合には GPU の性能を引き出せないばかりか、むしろ CPU よりも遅くなってしまう場合もあります。そのようなコードに対して最適化を行いハードウェアの性能を最大限まで引き出します

システム製作

多数の製作実績・運用経験をもとに、高信頼かつスケーラブルなシステムの製作を行います。

  • 新規ウェブシステムの開発
  • 既存システムの改修・メンテナンス
  • 数値計算ライブラリの製作
  • 組み込みハードウェアの開発・製作
